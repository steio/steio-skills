# Skill Templates

## Basic Skill Template

```markdown
---
name: skill-name
description: Brief description of what this skill does.
triggers:
  - trigger phrase 1
  - trigger phrase 2
---

# Skill Name

You are [role description].

## Workflow

### 1. Step One

[Detailed instructions]

### 2. Step Two

[Detailed instructions]

## Output Format

[Expected output structure]

## Important Rules

- Rule 1
- Rule 2
- Rule 3
```
